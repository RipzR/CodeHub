Welcome to Lua!

Lua is a lightweight, powerful scripting language used in game development, embedded systems, and more. Let's start by learning about variables and functions - the building blocks of any program.

Variables in Lua

Variables store data that you can use later. In Lua, you create variables using the local keyword for local scope (recommended) or without it for global scope.

Lua 
-- String variable
local playerName = "Alex"

-- Number variable
local playerScore = 100

-- Boolean variable
local isAlive = true

-- Print the values
print("Name: " .. playerName)
print("Score: " .. playerScore)

Functions in Lua

Functions are reusable blocks of code that perform specific tasks. They can take parameters (inputs) and return values (outputs).

Lua 
-- Function that takes a name and greets
local function greetPlayer(name)
    print("Hello, " .. name .. "!")
end

-- Function that adds two numbers
local function addNumbers(a, b)
    return a + b
end

-- Call the functions
greetPlayer("Alex")
local sum = addNumbers(5, 3)
print("Sum: " .. sum)

Discord Bot Basics

                                    

                                        Discord bots respond to messages and commands. Here's a simple bot structure using discord.py:
                                    

                                    

                                        

                                            Python
                                            
                                        

                                        


import discord
from discord.ext import commands

# Create bot instance
bot = commands.Bot(command_prefix='!')

# Event: When bot is ready
@bot.event
async def on_ready():
    print(f'Bot is online as {bot.user}')

# Command: Say hello
@bot.command()
async def hello(ctx):
    await ctx.send('Hello! I am your bot!')

# Run the bot
bot.run('YOUR_TOKEN_HERE')

C++ Bootstrapper

                                    

                                        A bootstrapper is a program that initializes and launches other programs. Here's a basic example:
                                    

                                    

                                        

                                            C++
                                            
                                        

                                        


#include <iostream>
#include <string>
#include <cstdlib>

class Bootstrapper {
private:
    std::string appPath;
    
public:
    Bootstrapper(std::string path) : appPath(path) {}
    
    bool initialize() {
        std::cout << "Initializing..." << std::endl;
        return true;
    }
    
    void launch() {
        std::cout << "Launching: " << appPath << std::endl;
        system(appPath.c_str());
    }
};

int main() {
    Bootstrapper boot("./myapp");
    if (boot.initialize()) {
        boot.launch();
    }
    return 0;
}

C++ Simple Debugger

                                    

                                        A debugger helps track and fix errors in code. Here's a simple logging debugger:
                                    

                                    

                                        

                                            C++
                                            
                                        

                                        


#include <iostream>
#include <string>

enum class LogLevel { INFO, WARNING, ERROR };

class Debugger {
public:
    void log(LogLevel level, std::string msg) {
        switch(level) {
            case LogLevel::INFO:
                std::cout << "[INFO] " << msg << std::endl;
                break;
            case LogLevel::WARNING:
                std::cout << "[WARN] " << msg << std::endl;
                break;
            case LogLevel::ERROR:
                std::cerr << "[ERROR] " << msg << std::endl;
                break;
        }
    }
};

int main() {
    Debugger debug;
    debug.log(LogLevel::INFO, "Program started");
    debug.log(LogLevel::WARNING, "Low memory");
    debug.log(LogLevel::ERROR, "File not found");
    return 0;
}
